# PID_CAR —— 差速机器人双PID轨迹跟踪控制
**C++ 标准工程化实现 | 运动控制基础项目**

## 项目简介
本项目基于 C++ 实现**双PID闭环轨迹跟踪控制**，使差速驱动机器人从初始坐标 (0, 0) 精准移动到目标点 (3, 3)。
项目严格遵循工业级代码规范，采用 **头文件与源文件分离** 的工程结构，是运动控制、规控算法方向的经典入门实战项目。

## 核心技术亮点
 双PID控制器（线速度控制 + 角速度控制）  
 差速机器人运动学建模与位姿更新  
 抗积分饱和机制（防止超调与震荡）  
 控制量输出限幅（符合真实物理约束）  
 精准到达判断（距离误差 < 0.05m）  
 标准 C++ 工程结构（.h + .cpp 模块化）
